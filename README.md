# Sliding Puzzle game with Vue/Vuex

This is a sample project to try Vue/Vuex.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm test
```

### Lints and fixes files
```
npm run lint
```
